Apa itu CEIR dan mengapa penting?

CEIR (Central Equipment Identity Register) adalah database resmi Kemkominfo yang mencatat status legalitas perangkat telekomunikasi di Indonesia. Perangkat yang tidak terdaftar atau berstatus blacklist tidak dapat menggunakan jaringan seluler di Indonesia. Pastikan dahulu sebelum membeli.

Apakah data IMEI saya aman?

Ya, sangat aman. Kami menggunakan enkripsi SSL dan tidak menyimpan data IMEI Anda setelah pengecekan selesai. Data hanya digunakan untuk proses pengecekan.

Berapa lama proses pengecekan CEIR?

Proses pengecekan sangat cepat, biasanya selesai dalam waktu kurang dari 30-80 detik setelah melakukan pesanan. Hasil akan langsung ditampilkan di website.

Metode pembayaran apa saja yang tersedia?

Kami menerima pembayaran melalui QRIS. Semua pembayaran diproses otomatis.

Bagaimana cara membaca hasil Status CEIR?

Jika hasil Cek Roaming Status menunjukkan Roamer, maka IMEI perangkat dalam mode turis. Jika Anda bukan turis, maka dapat dipastikan Anda membeli perangkat ilegal, dan dalam kurun waktu tertentu sinyal akan hilang.


Jika hasil Cek Roaming Status menunjukkan Registered, maka dapat dipastikan IMEI perangkat resmi atau legal. Namun jika perangkat masih tidak mendapatkan sinyal, artinya kemungkinan hasil registrasi yang tidak resmi.


Jika hasil Cek Roaming Status menunjukkan UNKNOWN, maka dapat dipastikan IMEI perangkat ilegal dan juga tidak dalam mode turis.

Bagaimana cara membaca hasil History CEIR?

Jika Cek History CEIR tidak menunjukkan apa-apa, ada dua kemungkinan: kemungkinan pertama perangkat tidak resmi atau blacklist, kemungkinan kedua perangkat resmi atau sudah membayar pajak di Bea Cukai. Namun untuk Bea Cukai, Anda harus mengecek lagi Cek Roaming Status dan hasilnya harus registered baru bisa dipastikan perangkat tersebut resmi.


Jika Cek History CEIR IMSI memiliki nilai, maka perangkat tersebut adalah Resmi Kemenperin. Namun jika perangkat tidak memiliki sinyal, kemungkinan besar adalah hasil registrasi yang tidak resmi. Silakan pastikan ke tempat Anda membeli perangkat.


Jika Cek History CEIR memperlihatkan hasil seperti add_roamer, inbound_roamer, remove_roamer, sudah dipastikan perangkat tersebut IMEI sedang dalam mode turis. Jika Anda bukan turis, silakan pastikan ke tempat Anda membeli perangkat.
